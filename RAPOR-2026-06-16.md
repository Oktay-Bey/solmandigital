## 2. Yapılan İş (15 iterasyon, 30 commit — hepsi canlıda)

### A. Conversion ölçüm onarımı (en kritik)
- `transport_type: "beacon"` → event navigasyondan sağ çıkar (qualify_lead artık kaydolur)
- ContactForm / ContactFormEn / ExitIntentPopup → `qualify_lead` dönüşümüne bağlandı
- Tek-atış conversion modeli (çift sayım önlendi), event-adı tutarlılığı
- Calendly URL'deki BOM bug'ı (`/en/﻿https:/...` kırık route) temizlendi

### B. CRO (conversion rate optimization)
- **/fiyatlar** (%87 bounce): paket CTA'ları sayfa içi forma, risk-reversal güven şeridi, form çerçevesi
- **/ai-otomasyon-hizmeti** + **/ucretsiz-analiz**: split hero (sağda form), AILeadForm buton grid
- AuditForm: zorunlu alan azaltma (friction ↓)

### C. İçerik derinliği & iç linkleme
- Yeniden kullanılabilir `RelatedGuides` (light + dark tema, çoklu kategori fallback)
- Tüm ana landing + /fiyatlar + ~16 ilçe sayfası → ilgili rehber bloğu (bounce ↓, topical authority)
- rehber/[slug]: React key bug fix + mid-article CTA eşiği iyileştirme

### D. WhatsApp ölçümü (%100 site geneli)
- `WhatsAppLink` component'i (beacon + source bazlı) → tüm inline WhatsApp tıklamaları izleniyor

### E. Pipeline koruması
- Email API rate limiting (`lib/rate-limit.ts`, 5 istek/60sn/IP) — form spam + Brevo/Resend kota suistimali

### F. Google Ads (re-auth + optimizasyon)
- Eksik `/api/google-ads/callback` OAuth route'u eklendi → re-auth tamamlandı, yeni refresh_token uygulandı
- `setGeoTargetType` eklendi (PATCH `geoTargetType` desteği)

## 3. Google Ads Analizi & Canlı Aksiyonlar

**Kampanya "Arama 2025" (23914856579), 30g:** 689 TL, %7.2 CTR (güçlü), 0 ölçülen dönüşüm.

**Bulgu:** Top 100 search-term'in **~%50'si (249 TL) İngilizce/bilgi-amaçlı** sorgu (how to / best /
ai tools / iot) — alıcı niyeti yok. TR alıcı-niyetli sadece 64 TL (6 terim).

**Uygulanan (canlı hesapta):**
1. **22 broad negatif keyword** — how to, what is, tutorial(s), course(s), examples, ideas, guide,
   learn, jobs, salary, career, internet of things, iot, tools, systems, vs, top 10, list of, free download, meaning
2. **Presence-only targeting** — yalnızca fiziksel olarak Türkiye'de olanlara gösterim
   (düşük bütçede "interest" kaynaklı global trafik israfı kesildi)

Çift-anlamlılar (best/platform/ai/software) kasıtlı dışarıda bırakıldı — EN buyer-intent +
"saas platform geliştirme" gibi TR terimleri bloklanmasın.

## 4. Kimlik Doğrulama Notu (kalıcı referans)
- **Google Ads = OAuth** (refresh_token). Google Ads API service account'u (standart hesapta)
  desteklemez → OAuth tek ve doğru yöntem.
- **GA4 = service account** (`GOOGLE_SERVICE_ACCOUNT_JSON`).
- Kurulum doğru; sorun yöntem değil, ölçümün hiç kaydedilmemesiydi.

## 5. Açık / İzlenecek (loop kapandı — manuel takip)

1. **GA4 `qualify_lead > 0` doğrulaması** — beacon fix canlı; 24-48h içinde dönüşümler kaydolmalı.
   GA4 → Etkinlikler → `qualify_lead` sayısı kontrol et.

3. **Sonraki optimizasyon** (veri birikince): kanıtlanmış TR terimleri (web tasarım fiyatları,
